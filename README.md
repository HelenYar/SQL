Задание 1 (практика)

Спроектировать схему (таблицы и связи между ними) для музыкального сайта. Требования:

- На сайте должна быть возможность увидеть список исполнителей.
- Для каждого исполнителя можно получить список его альбомов.
- Для каждого альбома можно получить список треков, которые в него входят.
- У исполнителя есть имя (псевдоним).
- У альбома есть название и год выпуска.
- У трека есть название и длительность.
- Сборник имеет название и год выпуска. В него входят различные треки из разных альбомов.
- Исполнители могут петь в разных жанрах, как и одному жанру могут принадлежать несколько исполнителей.
- Альбом могут выпустить несколько исполнителей вместе. Как и исполнитель может принимать участие во множестве альбомов.
- Трек входит только в один альбом, как и альбом принадлежит только одному исполнителю
